# Security Policy

## Supported Versions

Only the latest released version of `gixen-cli` receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in `gixen-cli`, please report it
privately by emailing **hsukenooi@googlemail.com**. Do not open a public
GitHub issue.

Because `gixen-cli` handles credentials (`GIXEN_USERNAME` / `GIXEN_PASSWORD`),
please give us a chance to issue a fix before disclosing the issue publicly.

You can expect an initial response within a few days. Once the issue is
confirmed and a fix is available, we will coordinate disclosure with you.
