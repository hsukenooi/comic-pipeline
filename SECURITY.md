# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email [hsukenooi@googlemail.com](mailto:hsukenooi@googlemail.com) with a description of the vulnerability and steps to reproduce it. You can expect a response within 7 days.
